# 公交车管理系统

这是一个基于Vue 3 + TypeScript + Vite的公交车管理系统前端项目。

## 功能特性

- 🚌 车辆管理 - 管理公交车信息
- 👨‍✈️ 司机管理 - 管理司机信息  
- 🗺️ 路线管理 - 管理公交线路
- 📅 调度管理 - 调度安排和日历视图
- 📊 统计分析 - 数据仪表板
- 🗺️ 实时监控 - 实时地图监控
- 🔐 用户认证 - 登录认证系统

## 技术栈

- **前端框架**: Vue 3
- **编程语言**: TypeScript
- **构建工具**: Vite
- **UI框架**: Tailwind CSS
- **状态管理**: Pinia
- **路由**: Vue Router
- **图标**: Vue Icons
- **地图**: 高德地图API

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发环境运行

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览构建结果

```bash
npm run preview
```

## 项目结构

```
src/
├── api/          # API接口
├── assets/       # 静态资源
├── components/   # 公共组件
├── composables/  # 组合式函数
├── lib/          # 工具库
├── router/       # 路由配置
├── stores/       # 状态管理
├── types/        # TypeScript类型定义
├── utils/        # 工具函数
└── views/        # 页面组件
```

## 开发说明

该项目使用现代化的前端技术栈，采用组件化开发模式，支持响应式设计，适配各种设备屏幕。
